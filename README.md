# Pedunoch Foods, home page design preview

A single-file design preview for the Pedunoch Foods e-commerce build, prepared by Curiosity.

**Live preview:** https://adesojidapo.github.io/pedunoch-demo/

## What this is

One self-contained `index.html` (fonts and photos embedded, no external requests) showing the proposed home page direction:

- Knockout hero: product photography masked inside giant PEDUNOCH letterforms
- Handwritten script wordmark and accents (Sacramento, open licence)
- Brand palette taken from the Pedunoch Instagram graphics: market orange on warm cream
- Seasonal announcement card, script watermarks, hand-drawn food outline doodles (pepper, onion, snail, prawn) and a Nigeria line-art outline
- Staples grid with unit pricing (per mudu, per paint, per litre)
- Shopping-list flow and quote-first international ordering, both mapped to how the business already operates

## Notes

- Prices are sample placeholders for the preview only.
- Interim photography: Wikimedia Commons contributors (CC BY-SA). To be replaced by the brand's own photography in the production build.
- The production build is a Next.js + Supabase + Paystack shop; this preview covers design direction only.
